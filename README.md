# ENTROPY


ENTROPY — the brand

The Entropy Collection —  limited edition of garments as a body of work

Entropy Jacket №01 to №21  — each individual garment 

The Jacket Protocol — the governing document

Proof of Work/Wear — the descriptive subtitle of the project

Built to be Worn — the footer line, the reinforcing statement

This is a clean hierarchy. Each layer does different work. The brand names the thing. The protocol names the rules. The subtitle names the concept. The footer states the purpose.
